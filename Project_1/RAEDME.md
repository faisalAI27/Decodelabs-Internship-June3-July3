Project Title: Rule-Based AI Chatbot

Description:
This is a simple rule-based chatbot built using Python. It responds to predefined user inputs using control flow, dictionary lookup, and if-else logic.

Features:
- Handles greetings
- Handles exit commands
- Runs in a continuous loop
- Cleans user input
- Gives fallback response for unknown input

How to Run:
python chatbot.py
